# Pixel in the Dungeon

Pixel in the Dungeon es un videojuego 2D tipo roguelike desarrollado con Godot Engine.  
El juego genera sus niveles de forma procedural, ofreciendo partidas distintas en cada intento y un sistema de combate rápido y dinámico.

Este proyecto forma parte de un Trabajo Final de Grado centrado en el desarrollo de videojuegos y en el uso de generación procedural para crear entornos jugables de forma automática.

---

## Gameplay

El jugador explora una mazmorra generada proceduralmente mientras combate contra distintos enemigos.  
Cada partida es diferente gracias al sistema de generación de niveles, que distribuye de forma aleatoria enemigos, objetos y mejoras.

El objetivo es avanzar entre niveles derrotando enemigos y mejorando las habilidades del personaje.

---

## Características

- Generación procedural de niveles
- Combate rápido con disparo direccional
- Diferentes tipos de enemigos
- Sistema de mejoras entre niveles
- Estilo visual pixel art
- Alta rejugabilidad

## Instalación y ejecución

1. Descargar el repositorio.
2. Abrir el proyecto con **Godot Engine 4.5**.
3. Ejecutar el proyecto desde el editor.

También se puede descargar el ejecutable desde la sección **Releases**.
